# pressureblush
PressureBlush presents a real-time deformation-to-appearance framework that maps self-contact neural deformation signals to shader-computed blanching and hyperemia, producing physiologically motivated skin-color responses without additional runtime simulation, precomputed textures, or artist-authored masks.
